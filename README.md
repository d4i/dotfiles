dotfiles
========

Profiles of Zsh, Vim, etc.

Instration
----------

```sh
cd
git clone https://github.com/d4i/dotfiles.git
sh setup.sh
```

Attention: To allow the new links, setup.sh forces the following files to be removed if they exist.

* ~/.zshrc
* ~/.zshenv
* ~/.zprofile
* ~/.vimrc
* ~/.gemrc
